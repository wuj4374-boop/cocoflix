# CocoFlix 使用说明

CocoFlix 是一个私人流媒体影院系统，聚合多个资源站，提供统一的搜索和播放体验。

---

## 快速开始

### 1. 访问网站

打开前端地址：`https://cocoflix-xi.vercel.app`

### 2. 登录

默认管理员账号：
- 用户名：`admin`
- 密码：`admin123`

> 首次使用建议修改密码（个人中心 → 修改资料）。

### 3. 注册新账号

点击右上角「登录」→ 切换到「注册」标签 → 输入用户名和密码（邮箱可选）。

---

## 功能说明

### 搜索影片

1. 点击顶部导航栏的搜索图标（或按 `/` 键快捷打开）
2. 输入影片名称（支持电影、电视剧、动漫）
3. 按回车或点击搜索按钮
4. 搜索结果会显示来自多个资源站的聚合结果

### 查看详情

点击任意影片封面，进入详情页：
- 影片基本信息（评分、年份、类型、简介）
- 播放源列表（来自不同资源站）
- 剧集列表（电视剧/动漫）
- 演员阵容
- 相似推荐

### 播放影片

1. 在详情页点击「立即播放」
2. 播放器支持：
   - 画质切换（4K/1080P/720P/480P）
   - 源切换（如果某个源播放失败，切换到其他源）
   - 剧集切换（电视剧/动漫）
   - 键盘快捷键：空格暂停/播放，左右箭头快进/快退，F 全屏
3. 播放进度自动保存，下次打开自动续播

### 收藏影片

- 在详情页或播放页点击「收藏」按钮
- 在顶部导航栏 → 收藏，查看所有收藏的影片
- 支持取消收藏

### 观看历史

- 在顶部导航栏 → 历史，查看观看记录
- 显示观看进度条和续播按钮
- 支持删除单条记录或清空全部

### 个人中心

- 查看和修改个人资料
- 上传头像

| 按键 | 功能 |
|------|------|
| `/` | 打开搜索 |
| `空格` | 播放/暂停 |
| `←` | 快退 5 秒 |
| `→` | 快进 5 秒 |
| `F` | 全屏切换 |
| `Esc` | 退出全屏/关闭搜索 |

## 常见问题

### Q: 搜索没有结果？

A: 资源站的可用性会变化。如果搜索没有结果：
- 换一个关键词试试
- 等几分钟后重试（资源站可能临时不可用）
- 检查后端服务是否正常运行

### Q: 视频播放不了？

A: 如果某个源播放失败：
- 在播放页底部点击「切换源」，选择其他资源站
- 不同资源站的可用性不同，多试几个
- 部分资源有时效性，过期后需要换源

### Q: 页面加载很慢？

A: 首次访问时，后端可能在冷启动（免费方案限制），等待 30 秒后刷新即可。后续访问会快很多。

### Q: 如何防止后端休眠？

A: Render 免费计划 15 分钟无请求会休眠。可以使用 cron-job.org 定时 ping 后端的 `/api/v1/health` 接口，每 14 分钟一次。

### Q: 如何更新代码？

A: 推送代码到 GitHub 后，Vercel 和 Render 会自动重新部署。
